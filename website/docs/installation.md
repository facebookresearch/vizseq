VizSeq requires **Python 3.6+** and runs on **Unix/Linux**, **macOS/OS X** and **Windows**.

You can install VizSeq from PyPI repository:
```bash
$ pip install vizseq
```
Or install it from source:
```bash
$ git clone https://github.com/facebookresearch/vizseq
$ cd vizseq
$ pip install -e .
```
